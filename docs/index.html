Medical Image Analysis API

A RESTful API for analyzing medical images to detect various eye conditions including Diabetic Retinopathy (DR), Age-related Macular Degeneration (AMD), and Glaucoma.

Authentication

This API currently does not require authentication. All endpoints are publicly accessible.

Endpoints

POST

/api/v1/health_assessment

Analyze a medical image and return health assessment results.

Request

Content-Type: multipart/form-data

Parameters
Parameter Type Required Description
image file Yes Medical image file (JPEG, PNG, etc.)
actual_age integer Yes Patient's actual age (0-120)
Response

Content-Type: application/json

Success Response (200 OK)
{
  "DR": {
    "score": 2,
    "confidence": 0.92,
    "probabilities": [0.01, 0.07, 0.82, 0.09, 0.01]
  },
  "AMD": {
    "score": 1,
    "confidence": 0.85,
    "probabilities": [0.1, 0.8, 0.1]
  },
  "Glaucoma": {
    "score": 0,
    "confidence": 0.97,
    "probabilities": [0.97, 0.03]
  },
  "Biological age": 45,
  "report_image": "base64_encoded_image_string"
}
Error Responses
Status Code Description
400 Bad Request Invalid request parameters or missing required fields
404 Not Found Requested resource not found
405 Method Not Allowed HTTP method not allowed for the requested endpoint
500 Internal Server Error Server encountered an error while processing the request
Example Request
curl -X POST "http://localhost:5000/api/v1/health_assessment" \
  -H "accept: application/json" \
  -H "Content-Type: multipart/form-data" \
  -F "image=@path/to/image.jpg" \
  -F "actual_age=45"
import requests

url = "http://localhost:5000/api/v1/health_assessment"
files = {'image': open('path/to/image.jpg', 'rb')}
data = {'actual_age': 45}

response = requests.post(url, files=files, data=data)
print(response.json())
const formData = new FormData();
formData.append('image', document.querySelector('input[type=file]').files[0]);
formData.append('actual_age', 45);

fetch('http://localhost:5000/api/v1/health_assessment', {
  method: 'POST',
  body: formData
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Response Codes

Status Code Description
200 OK Request was successful
400 Bad Request Invalid request parameters or missing required fields
404 Not Found Requested resource not found
405 Method Not Allowed HTTP method not allowed for the requested endpoint
500 Internal Server Error Server encountered an error while processing the request

Error Handling

In case of an error, the API will return a JSON object with the following structure:

{
  "error": "Error type",
  "message": "Human-readable error message"
}

Common Error Responses

  • Missing required fields 
    {
  "error": "Bad Request",
  "message": "Missing required parameter: image"
}
  • Invalid file type 
    {
  "error": "Bad Request",
  "message": "Invalid file type. Only JPEG, PNG are allowed."
}
  • Invalid age 
    {
  "error": "Bad Request",
  "message": "Invalid age. Must be between 0 and 120"
}
  • Server error 
    {
  "error": "Internal Server Error",
  "message": "An unexpected error occurred while processing your request"
}
